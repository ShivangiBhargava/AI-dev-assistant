## New Features

- Download Report as PNG (frontend-only)
  - A new **Download PNG** button appears in the results panel header. It snapshots the visible results panel and downloads a PNG image (uses html2canvas). This is handy for sharing analysis output with teammates or embedding in documentation.

(If you enable this feature in the UI, no backend changes are required. If you prefer a server-side rendering option, we can discuss adding a proxy endpoint in a follow-up issue.)
